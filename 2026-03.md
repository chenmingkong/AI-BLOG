# AI资讯 2026年3月

## 1. WordPress.com现在允许AI代理撰写和发布文章

- **平台**: TechCrunch
- **作者**: Sarah Perez
- **发布时间**: 2026-03-20
- **分类**: AI应用
- **收藏时间**: 2026-03-20
- **原文链接**: https://techcrunch.com/2026/03/20/wordpress-com-now-lets-ai-agents-write-and-publish-posts-and-more/

**摘要**: Web托管平台WordPress.com宣布将允许AI代理在客户网站上撰写、编辑和发布内容，以及管理评论、更新和修复元数据、组织内容标签和分类等。通过自然语言命令控制界面，网站几乎可以完全由AI代理创建和运行。此举可能改变互联网的外观和感觉，降低网站设置和维护的门槛，但也可能让网络充满不再由人类撰写的、由机器生成的内容。WordPress为互联网上超过43%的网站提供动力，每月有200亿页面浏览量和4.09亿独立访客。

## 2. NVIDIA宣布推出NemoClaw，为OpenClaw社区带来安全和隐私

- **平台**: Nvidia News
- **作者**: NVIDIA
- **发布时间**: 2026-03-16
- **分类**: AI基础设施
- **收藏时间**: 2026-03-20
- **原文链接**: https://nvidianews.nvidia.com/news/nvidia-announces-nemoclaw

**摘要**: NVIDIA宣布推出NVIDIA NemoClaw堆栈，用于OpenClaw代理平台。该堆栈允许用户通过单一命令安装NVIDIA Nemotron模型和 newly announced NVIDIA OpenShell运行时，为自主AI代理添加隐私和安全控制。NemoClaw使用NVIDIA Agent Toolkit软件优化OpenClaw，安装OpenShell提供开放模型和隔离沙箱，为自主代理添加数据隐私和安全性。Nvidia CEO Jensen Huang表示："Mac和Windows是个人电脑的操作系统。OpenClaw是个人AI的操作系统。这是行业一直在等待的时刻——软件新时代的开始。"
